# Editor Agent（編集提案エージェント）

生成された原稿を分析し、改善提案を行うエージェントです。

## 機能

1. **繰り返し表現の検出**: 頻出する表現を特定
2. **文の長さチェック**: 長すぎる文を検出
3. **段落バランス分析**: 段落の長さのばらつきをチェック
4. **文末表現の多様性**: 単調な文末パターンを検出
5. **セクション構成分析**: 適切な構成になっているか確認
6. **導入の強さ評価**: 冒頭のインパクトを評価

## 使用方法

### オーケストレーター経由（推奨）

```bash
node orchestrator/cli/run.js -p "プロジェクト名"
```

### 直接実行

```javascript
const { execute, analyzeForEditing } = require('./index');

const result = await execute({
  content: '原稿テキスト...'
}, {
  projectPath: '/path/to/project'
});

console.log(result.output.suggestions);
```

## 提案項目

| チェック項目 | 説明 |
|-------------|------|
| 繰り返し表現 | 4回以上使用された表現 |
| 文の長さ | 100文字以上の文 |
| 段落バランス | 平均の2倍/0.3倍を超える段落 |
| 文末表現 | 50%以上同じ文末パターン |
| セクション構成 | 3〜8セクションが推奨 |
| 導入の強さ | 50文字以上、問いかけ/断定の有無 |

## 出力例

```json
{
  "suggestions": [
    {
      "title": "繰り返し表現",
      "description": "「という」「している」などの表現が繰り返されています。類義語への置換を検討してください。"
    }
  ]
}
```

## レポート出力

`reviews/v{version}_suggestions.md` にMarkdown形式で保存されます。
